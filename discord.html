BCA Discord

From this widget, you can view the messages in all public groups. You will also be able to post without having a discord account.

Rules

  • Please limit any spam/bot commands to #spam.
  • Set your nickname to something identifiable (with /nick).

Info

  • Suggestions
  • MathBot
  • Text Formatting

Suggestions

  • Go to #requests and type !suggest <your suggestion here> to make a suggestion
  • The suggestion will show up in #announcements
  • You will then be able to ⬆️ (upvote) and ⬇️ (downvote).

MathBot

  • =about: About MathBot
  • =blame: See who fired MathBot
  • =calc: Basic calculator (aliased to ==)
  • =roll: Roll dice
  • =help: Get help
  • =tex: Output LaTeX
  • =wolf: Queries Wolfram|Alpha

Text Formatting

  • *italics*: italics
  • **bold**: bold
  • _underline_: underline
  • ~strikethrough~: strikethrough
  • ```code block```: code block
  • And you can mix & match them.
